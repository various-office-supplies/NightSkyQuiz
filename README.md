# Night Sky Quiz

> [!NOTE]
> Requires Digistar 6.  Learn more <a href="https://www.es.com/digistar/">here</a>.

Created by Ben Kussmann in collaboration with Kevin Healy and James Enos.

[Come visit the planetarium!](https://www.mesacc.edu/departments/physical-science/astronomy/planetarium/free-family-shows) 

<br>

# Overview
This Digistar plugin creates quizzes for students to test their knowledge of the night sky!
- replace hand-written .ds scripts with an intuitive GUI
- customize with a simple configuration file, saving hours of work for each quiz
- set any custom date + location
- order items by group, alphabetically, or manually with `!order`
- add any native DS commands, such as `planetAtmosphere`, `aurora`, or `eclipticText intensity`
- compatible with all common DS objects (constellations, stars, directions, etc.)

<br>

### Installation:
You can download this GitHub repository, or search for us directly in Digistar ("Night Sky Quiz" by user `Mesa`):

<img width="623" height="462" alt="Install the Plugin" src="https://github.com/user-attachments/assets/673ecf10-1db1-4e5f-a079-0714eaa0683a" />

<br><br>

Once opened, run the program and click "Start Quiz", then select `sample quiz.txt` from the project folder (by default, located in `$Content\User\Downloads\Mesa\NightSkyQuiz\`).
Once loaded, the quiz should look something like this:

<img width="1920" height="1165" alt="only a few items selected" src="https://github.com/user-attachments/assets/b3b90472-a01d-4aff-b843-ad5083e7a97e" />

<br><br>

Go ahead and select a few items from the console.  On your planetarium's screen, you should see the quiz items appearing on-screen, labeled with the number enabled:

<img width="756" height="567" alt="example of quiz output" src="https://github.com/user-attachments/assets/02bfdec2-7b17-4c23-82a7-b611459f7fb9" />

<br><br>

... and that's it! To customize your quiz, please see the instructions at the bottom of `sample quiz.txt`.
